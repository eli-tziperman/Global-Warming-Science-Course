DESCRIPTION:
This data set contains data derived from the Quelccaya Summit Ice 
Core by Drs. Lonnie Thompson and Ellen Mosley-Thompson of The 
Ohio State University.  The data was updated in May 1992 to extend 
the data period of record.  A printed documentation volume is now 
available from NGDC.  The data contained in the files 'Q83SUM.ALL' 
and 'Q83COR1.ALL' represent the annual information for the summit 
core and core 1, drilled in 1983.  The data for the year 1984 are from a 
pit excavated in the year following the drilling and thus the 1984 data 
are the same for both the summit core and core 1.  The columns in the 
tables are as follows:

Column 1: Thermal Year
Columns 2,3, and 4: Particulate concentrations are measured using the 
Coulter counter 	technique and are total concentrations per milliliter 
sample for the following size 	ranges: (in order from left to right): 
Particles with diameters from 0.63 to 0.83um, particles with 
diameters  < 0.63 um, Particles with diameters > 1.59 um
Column 5: Conductivity in uS CM-1
Column 6: Oxygen Isotopic Ratios (o/oo)
Column 7: Net annual accumulation (Meters of ice equivalent) Note: This file corrected 4/20/2015, previously column 7 was incorrectly labeled as Meters of Water equivalent. 
Column 8: Net annual accumulation as standard deviations from the 
complete series mean.
